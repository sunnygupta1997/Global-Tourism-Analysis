🌍 Tourism Insights Dashboard (Power BI)

 📊 Project Overview

This project focuses on analyzing tourism data to uncover meaningful insights related to travel behavior, revenue generation, and customer satisfaction.
The dashboard is built using Power BI and provides an interactive way to explore key metrics.


🛠 Tools & Technologies

* Power BI
* Microsoft Excel


📁 Dataset Description

The dataset includes:

* Tourist demographics (Age, Gender, Nationality)
* Travel details (Destination City, Travel Purpose, Transport Mode)
* Accommodation information (Hotel, Stay Duration)
* Financial data (Total Trip Cost, Payment Mode)
* Feedback Score (Customer Satisfaction)


📈 Key Insights

* Total number of tourists and overall revenue generated
* Most popular destination cities
* Tourist distribution by nationality
* Travel purpose analysis (Leisure vs Business)
* Preferred transport and accommodation types
* Customer satisfaction based on feedback score


📊 Dashboard Features

* Interactive slicers for dynamic filtering
* KPI cards (Total Tourists, Total Revenue, Avg Feedback)
* Bar charts, Pie charts, and trend analysis
* Clean and user-friendly layout



🎯 Business Value

This dashboard helps:

* Identify high-performing tourist destinations
* Understand customer preferences
* Improve marketing and travel strategies
* Enhance customer experience


 🚀 How to Use

1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Use filters to explore different insights
